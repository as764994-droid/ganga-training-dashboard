# 📊 Project GANGA — Training Dashboard

**Live Dashboard →** `https://YOUR-USERNAME.github.io/ganga-training-dashboard`

A responsive, interactive training session dashboard for **Project GANGA** (Government Assisted Network for Growth & Advancement) — Uttar Pradesh Government's broadband entrepreneurship initiative.

---

## 🚀 How to Use

1. Open the dashboard link above
2. Click **"Upload Training Excel"**
3. Select the Excel file for any batch
4. Instantly view charts, KPIs, and breakdowns

Works on **mobile and desktop**. No login required.

## 📈 Dashboard Features

- **KPI Cards** — First Batch total, Confirmed, Calls Connected, Denied, Next Batch, Session Presence
- **Confirmation Breakdown** — Donut chart by WhatsApp / Email / Both
- **Gender Split** — All leads vs Confirmed only
- **Final Remarks** — Horizontal bar, all categories
- **District-wise Leads** — Stacked bar by status
- **KAM Breakdown** — Full performance per KAM with confirmation rates
- **Lead Table** — Searchable, filterable full data table
- **Filters** — By KAM, District, Status

---

## 🛠️ Tech Stack

Pure HTML · CSS · JavaScript · [Chart.js](https://www.chartjs.org/) · [SheetJS](https://sheetjs.com/)
No framework, no backend, no dependencies to install.
